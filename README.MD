TinyML project for HWR.

Hardware:
1 * Arduino Nano 33 BLE Sense REV2
1 * Micro-USB-Cable

Libs:
Library Arduino_BMI270_BMM150 required.

Borad Manager:
Arduino Nano Mbed OS Nano Boards required.

External:
https://github.com/tensorflow/tflite-micro-arduino-examples
